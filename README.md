# Sales Analyzer — MVP (Streamlit)

Application Streamlit de type “Data Analyst Assistant” pour analyser rapidement un fichier de ventes (CSV) :
- ingestion robuste
- audit qualité automatique
- nettoyage contrôlé (simulation + application)
- mapping sémantique (Date/Montant/Produit/Client/Canal)
- KPIs & dashboard direction
- exports COMEX : CSV / Excel / PDF
- comparatif YoY (N vs N-1) avec fallback automatique si historique insuffisant

## Prérequis
- Python 3.13
- macOS / Windows / Linux
- (recommandé) environnement virtuel `.venv`
